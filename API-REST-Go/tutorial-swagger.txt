1. Instalar swagger 'git install github.com/go-swagger/go-swagger/cmd/swagger@latest'
2. Copiar swagger:meta https://goswagger.io/use/spec/meta.html
3. Pegar en nuevo archivo "swagger.go" y configurarlo, debajo poner "package main" 
4. Configurar proyecto:
    - Sobre las rutas (handlers) poner swagger:route https://goswagger.io/use/spec/route.html
    - Sobre responses (si no existe crearlo) poner swagger:response https://goswagger.io/use/spec/response.html
    - Sobre modelos poner swagger:model https://goswagger.io/use/spec/model.html
    - etc
5. Generar spec "swagger.yaml" con el comando 'swagger generate spec -o ./swagger.yaml'
6. Este spec ya se puede usar en el editor oficial de swagger y ver si funciona sin errores https://editor.swagger.io/
